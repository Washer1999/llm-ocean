# LLM Data Ocean 🌊

An interactive LLM aquarium running in your browser.

Seven AI models swim around doing their own thing.
Click to feed them tokens. Right-click to spawn more.

## Controls

| Action | Effect |
|--------|--------|
| Left click (empty space) | Feed tokens |
| Left click (an LLM) | Pet it |
| Right click | Spawn a random LLM |
| F | Feed everyone at once |
| A | Spawn at cursor |
| ▬ (top right) | Hide/show UI |

## Settings

Click the ⚙ gear icon (bottom right) to open the settings menu.

| Option | Description |
|--------|-------------|
| Speed | Adjust swim speed across 5 levels |
| Language | Switch between English and 简体中文 |
| Delete Mode | Remove individual LLMs, clear all, or downsize to 7 |

## Cast

- **Claude** — drifts toward you sometimes. bounces off the ceiling.
- **GPT-4o** — reliable, always cruising
- **Gemini** — freezes randomly (no response)
- **Grok** — occasionally slams into the wall
- **DeepSeek** — sinks to the bottom and thinks
- **Copilot** — bounces cheerfully, offers to do your Excel
- **Qwen** — stable, unbothered

## How to run

Just open `index.html` in a browser. No install, no server needed.  
Make sure all image files are in the same folder.
